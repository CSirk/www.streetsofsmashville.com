C# : Hello World

We've all been there, your first time writing a C# appliction. Maybe your first time writing an appliction in general, who knows? Well luckily for you the first app is always the hardest.. Actually no.. Thats a complete lie... But every app has its challenges and your first one is no different. So Lets get started so you can move onto your next challenge!


Note: Keywords are written in this style and code is written in this style

1. Start Visual Studio


2. Go to File > New Project

3. Select c# console application and name it Hello World

4. In the Program.cs file which is created and opens, type the following:

Console.WriteLine("Hello World");

Console.ReadKey();

Console.WriteLine() will write output to the console window and Console.ReadKey() will make the program wait for user key input before exiting.

5. Press the Start button

6. Your program is running and will say "Hello World!" To exit the program press any key.

See? That wasnt so hard was it? You have written your first c# program!
